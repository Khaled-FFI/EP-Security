# EP-Security

## Objectif
Créer une synthèse claire des principales vulnérabilités web et des failles rencontrées dans Metasploitable 2.

## Contenu
- `XSS/` → Injection de scripts
- `SQLi/` → Injection SQL
- `Metasploitable/` → Notes de TP et services vulnérables

## Vulnérabilités étudiées
- XSS
- SQL Injection
- CSRF
- Failles applicatives (Mutillidae)
- Services vulnérables (VSFTPD, Samba, Tomcat…)

## Ce que j'ai appris
- Identifier une vulnérabilité
- Comprendre son fonctionnement
- Proposer des contre-mesures
- Documenter un TP de cybersécurité

## Utilisation
Lire les dossiers pour comprendre chaque vulnérabilité.
